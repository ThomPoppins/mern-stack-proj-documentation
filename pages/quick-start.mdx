### Backend Configuration

1. Navigate to the `/backend` folder in your file explorer.
2. Create a `config.js` file.
3. Add the following constants and update them with your personal values:

   ```javascript filename="Configuration: /backend/config.js" {3} copy
   export const PORT = 5555;
   export const mongoDBURL = "your-mongodb-connection-url";
   export const JWT_SECRET = "your-secret-key";
   export const KVK_TEST_API_KEY = "your-test-api-key";
   export const KVK_PROD_API_KEY = "your-prod-api-key";
   ```

### Set Up Frontend Configuration

1. Navigate to the `/frontend` folder.
2. Create a `config.js` file if it doesn't exist.
3. Add the following constant and export it:

   ```javascript filename="Configuration: /frontend/config.js" {3} copy
   export const BACKEND_URL = "http://localhost:5555";
   export const TEST_KVK_API = false;
   ```

### Install Dependencies

- Inside the `/backend` folder, run:

  ```bash copy
  npm install
  ```

- Inside the `/frontend` folder, run:

  ```bash copy
  npm install
  ```

### Start Servers

- Inside the `/backend` folder, run:

  ```bash copy
  npm run dev
  ```

- In a separate terminal, inside the `/frontend` folder, run:

  ```bash copy
  npm run dev
  ```

### Access the Application

Visit the web application in your browser using the link provided by the Vite.js server after starting the frontend server.

Now you have the application up and running locally!
